# Mission: BMI Calculator Web App
Build a fast, no-signup BMI calculator to capture a share of 2.24M monthly searches.

## Phase 1: Build & Design
1. Inputs: Standard (ft/in/lbs) and Metric (cm/kg) toggles.
2. Output: A visual, color-coded gauge (using a lightweight JS canvas or CSS) showing Underweight, Normal, Overweight, Obese.
3. Monetization: Premium health RPM ad networks + affiliate links for fitness apps or equipment.

## Phase 2: Deployment
1. GitHub to Vercel deployment.
2. Implement strict accessibility (a11y) standards (ARIA labels) for SEO boosts.

## Phase 3: Ongoing Maintenance (Cron Jobs)
- **Weekly:** Run Lighthouse accessibility audit. Auto-fix any missing ARIA tags.
- **Monthly:** Scan competitor sites for new features (like "Healthy Weight Range" outputs) and auto-implement them via PR.
